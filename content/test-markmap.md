# Test Markmap - Mind Map Interactivo

Este es un ejemplo de diagrama interactivo usando Markmap. Puedes hacer zoom, pan y explorar la estructura jerárquica.

```markmap
# Desarrollo Web
## Frontend
### React
#### Hooks
#### Components
### Vue
### Angular
## Backend
### Node.js
#### Express
#### NestJS
### Python
#### Django
#### FastAPI
## DevOps
### Docker
### Kubernetes
### CI/CD
```

```markmap
# main
## title1
### love
  1. 1
  2. 2
  4. 4
  4. 4
  4. 4
  4. 4

## title2
```

## Características

- **Interactivo**: Zoom y pan con el ratón
- **Responsive**: Se adapta a diferentes tamaños de pantalla
- **Markdown**: Syntax directo desde markdown
- **Estilizado**: Estilos automáticos CSS

## Ventajas de Markmap

Markmap es perfecto para:
- Organizaciones jerárquicas
- Brainstorming
- Documentación estructurada
- Mapas mentales de proyectos
